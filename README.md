# Новое русское вино

Сайт магазина авторского вина "Новое русское вино".

## Запуск

- Скачайте код
- Установите зависимости командой `pip install pandas openpyxl jinja2`
- Запустите сайт командой `python3 main.py`
- Перейдите на сайт по адресу [http://127.0.0.1:8000](http://127.0.0.1:8000).

# Проверка работы

Проверка категорий товаров
На странице должны быть разделы:

Белые вина

Красные вина

Напитки

Проверка сортов винограда

У вин (белые и красные) указан сорт винограда

У напитков (коньяк, чача) сорт винограда не отображается

Проверка акций
На товарах с пометкой "Выгодное предложение" в Excel файле должен отображаться оранжевый значок в правом верхнем углу:

Белая леди (Белые вина) - есть значок

Чача (Напитки) - есть значок

Остальные товары - без значка
